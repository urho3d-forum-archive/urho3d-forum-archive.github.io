From the documentation on Urho2D:

if ‘seams’ between tiles are obvious then you should make your tilesets images nearest filtered (see Static sprites section above.)

You need to create an XML file in the same folder as the texture, with the same name (but ending in xml) containing:

<texture>
    <filter mode="nearest" />
</texture>
