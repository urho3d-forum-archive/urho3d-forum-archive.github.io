OverLib

OvermindDL1

I have been pulling out a lot of code that I have been sharing between my projects in to a library that they all use and have been using it for a few months now, it is not big yet as I still need to pull out more in to it (especially the Input Mapper), but it is still useful even as small as it is. It currently contains a few helpers for the primary Application, an AttributeEditor UI creator for many of the attribute types, and an Urho3D style event driven state manager with loading screen support. Check the project on GitHub for a fairly sizable README.md for detailed information: github.com/OvermindDL1/Urho3D-OverLib
